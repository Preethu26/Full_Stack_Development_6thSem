# Movie Search App 🎬

A simple and elegant movie search application using the OMDB API.

## Features

- Search for movies by title
- Display movie posters, titles, and release years
- Responsive design that works on all devices
- Beautiful gradient UI with smooth animations
- Enter key support for quick searches
- Error handling for failed requests

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+, Async/Await, Fetch API)
- OMDB API

## How to Use

1. Open `index.html` in your browser
2. Type a movie name in the search box
3. Click "Search" or press Enter
4. Browse through the movie results

## Setup

No installation required! Just open the `index.html` file in any modern web browser.

If you want to use the development server with nodemon:

```bash
npm install
npm run dev
```

## API

This project uses the [OMDB API](http://www.omdbapi.com/) to fetch movie data.
